### 11653. 소인수분해
(주소)https://www.acmicpc.net/problem/11653



#### 문제 요약:

숫자 N을 소인수분해 하는 문제이다.


#### 풀이 해설:



```c++
#include <iostream>
#include <algorithm>
#include <vector>
using namespace std;

int N;

int main(void) {
  cin>>N;

  // 소인수분해
  for(int i=2;i*i<=N;i++){
    while(N%i==0){
      cout<<i<<'\n';
      N/=i;
    }
  }
  if(N!=1) cout<<N;
}
```
